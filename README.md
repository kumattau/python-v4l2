# python-v4l2

A python ctypes binding for video4linux2

This binding contains the definitions
in the following headers of linux-libc-dev:amd64 5.4.0-81.91.

* asm-generic/ioctl.h
* linux/time.h
* linux/v4l2-common.h
* linux/v4l2-controls.h
* linux/videodev2.h
